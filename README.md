# 🏰 Dungeon Invaders

**Dungeon Invaders** é um jogo no estilo **Roguelike Survival**, onde o
jogador enfrenta **hordas de inimigos dentro de uma dungeon**,
adquirindo **poderes aleatórios** ao longo da partida para sobreviver o
máximo possível.

O objetivo é simples: **sobreviva, evolua e enfrente ondas cada vez mais
difíceis de inimigos.**

------------------------------------------------------------------------

# 🎮 Sobre o Jogo

Dungeon Invaders foi desenvolvido como um projeto acadêmico com foco em:

-   Mecânicas simples e acessíveis
-   Progressão rápida e recompensadora
-   Rejogabilidade através de poderes aleatórios
-   Combate contra hordas de inimigos variados

Cada partida é diferente, pois os **poderes adquiridos são gerados
aleatoriamente**, criando novas estratégias a cada tentativa.

------------------------------------------------------------------------

# 👤 Persona do Jogo

### Nome

Gabriel (persona fictícia)

### Idade

18--19 anos

### Perfil

Gabriel é um jovem estudante que gosta de videogames, mas não possui muito
tempo livre durante o dia por causa dos estudos e outras atividades.

Quando encontra um tempo para jogar, ele prefere **jogos rápidos,
simples de entender e que proporcionem sensação de progresso**, mesmo em
sessões curtas.

### Comportamento como jogador

-   Prefere jogos que possam ser jogados em **partidas curtas**
-   Gosta de **ver evolução do personagem rapidamente**
-   Se diverte enfrentando **inimigos diferentes**
-   Valoriza jogos **simples, mas desafiadores**
-   Prefere jogos que **não exijam longas histórias ou tutorial
    complexo**

### O que ele busca em um jogo

-   Progressão visível
-   Gameplay direto
-   Rejogabilidade
-   Desafio gradual
-   Experiência divertida mesmo em pouco tempo

Dungeon Invaders foi pensado para atender exatamente esse tipo de
jogador.

# ⚔️ Mecânicas Principais

-   Sistema de **hordas de inimigos**
-   **Poderes aleatórios (Power-ups)**
-   **Escalonamento de dificuldade**
-   Combate rápido e dinâmico
-   Alta **rejogabilidade**

------------------------------------------------------------------------

# 🧠 Sistema de Progressão

Durante o jogo, o jogador pode:

-   Derrotar inimigos
-   Sobreviver a ondas
-   Coletar **habilidades aleatórias**
-   Criar combinações únicas de poderes

Exemplos de poderes:

-   ⚡ Speed Boost\
-   💥 Double Shot\
-   ❤️ Health Regeneration

ferramenta.
